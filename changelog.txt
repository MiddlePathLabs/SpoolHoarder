Changelog - Recent Updates (May - July 2026)

## Version 2.4.0
[ Features ]
- Added an on-device inventory assistant that answers queries offline without using cloud credits.
- Surfaced on-device model availability status in the assistant.
- Added local parsing for favorite, glow, sealed, and barcode queries.
- Added local matching for locations, vendors, and printers.
- Added aggregate queries: total, average, min/max, distinct count, and share.
- Added within-last and older-than date filters with missing-data caveats.
- Added support for grouped and negated filters in assistant queries.
- Refreshed assistant help text and examples with new capabilities.

[ UI & UX ]
- Showed detected barcodes on batch scan review cards, including screen-reader announcements.
- Improved batch scan editing to use the full scanned spool data with brand suggestions drawn from your inventory.

[ Fixes ]
- Fixed assistant partial brand follow-ups.
- Fixed batch scan spool size defaults and review detail rendering.
- Kept inventory snapshot color pills on one line.

[ Security & Infrastructure ]
- Enabled the on-device assistant on Android devices.
