# SpoolHoarder

A cross-platform 3D printer filament inventory management app.

## Capabilities

### Inventory Management

* Track filament spools with comprehensive metadata (brand, material, color, dimensions, weight, price)
* Multi-color spool support with hex codes and color families
* Customizable fields for extruder/bed temperatures, flow ratios, and K-factors
* Purchase tracking with source, date, and price history
* Location tracking for inventory organization

### Organization \& Search

* Sort by brand, material, color, remaining weight, and more
* Filter by any attribute with quick-access facets
* Full-text search across all spool data
* Tag and group related spools
